# Finance Dashboard

Deployable static dashboard for bank transaction analysis.

How to use
1. Unzip and rename folder if needed.
2. Create GitHub repo and push contents, or drag & drop to GitHub web upload.
3. Activate GitHub Pages from Settings → Pages (branch: main, folder: /)

Features
- Upload CSV/JSON, auto-detect Czech bank columns
- Filters, tabs by transaction place, export CSV, shareable URL

Security
- Runs fully in browser, data never sent to server.
